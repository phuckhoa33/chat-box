Streamelements Chat Box Overlay - Installation Guide

Last Update 24 April 2022
This installation guide is for Streamelements web dashboard. Make sure you have downloaded the theme files from Gumroad.
Each themes might have different options. Not all steps are required or applicable to each themes.

Installation Steps

  1. Visit your Streamelements web dashboard and proceed to the overlay section.
  2. Click on Create New Overlay if you don't have an existing overlay or Edit to add the widget onto your existing overlay.
  3. Click on the Add Widget button or the + button, proceed to 'Static / Custom' section and click on 'Custom Widget'.
  4. Next, click on Open Editor button and replace everything with the files you downloaded from Gumroad, on their respective tab, HTML file on HTML tab, CSS file on CSS tab and so on.
  5. Click on 'Done' once you finished.
  6. A 'Customization' tab will appeared under the setting section. You can customize your chat widget under said section.
  7. You can edit your widget size under 'Position, size and style' section. You can ignore the 'Animation Settings'. That is it.
  8. You can now copy your overlay url and use it as a browser source in OBS.
