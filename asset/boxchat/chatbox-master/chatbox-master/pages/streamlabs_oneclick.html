Installation Steps

Video Version

Text Version

  1. Select one of the themes from 'Themes' section above.
  2. Upon clicking on it, a new tab will open directing you to Streamlabs' website.
  3. A popup will appear asking you where to install the chatbox as show in the image below. You can either use default/existing 'collection' or create new theme 'collection'.
  4. You will then be redirected to Streamlabs Chat Box widget page where all the settings and codes automaticly gets installed to.
  5. On this page, scroll down and click on Custom Fields section
  6. On this section, you can customize the chat box and make changes as you please.
  7. Finally, you can then copy your chat box widget url and use it on OBS.
