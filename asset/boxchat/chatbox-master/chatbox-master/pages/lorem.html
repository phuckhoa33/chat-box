( !Website Under Construction! ) Lorem Installation Guide

For version 1.0 - 20 December 2021
This installation guide is for Streamlabs web dashboard, not Streamlabs Desktop (SLOBS).

Streamlabs Settings

Visit your Streamlabs web dashboard and proceed to chat widget section. Setup your options as shown below.
  1. Themes Ignore
  2. Badges Choose as you please
  3. Extra Emotes Recommended to enable
  4. Background Color Ignore
  5. Text Color Ignore
  6. Font Size Ignore
  7. Hide Message After Always show messages
  8. Chat Delay Choose as you please
  9. Hide Chatter Recommended to disable
    10. Hide Chatter Option

    If you enable this option, later please make sure to use Lorem-JS-Hide-Chatter file instead of Lorem-JS. You must also enable both Hide Common Chat Bots & Hide commands starting with `!` option.

    If you use custom made/named bot, you have to find code snippet below in the JS file and add in your bot name(s).

    const botNames = ["nightbot","moobot","xanbot","deepbot","vivbot","phantombot","streamelements"];
  10. Muted Chatters Recommended to disable
  11. Custom Bad Words Recommended to disable
    12. These options should not effect anything. If you have any problems, please contact me.
  12. Enable Custom HTML/CSS Enabled

Lorem Settings

  1. After enabling Custom HTML/CSS, new field should appear like the image shown below.
  2. You can now open Lorem theme files (Lorem-HTML, CSS, JS & Custom-Field).
  3. Click on each tab and replace the content with their respective code. Lorem-HTML on the HTML tab, Lorem-CSS on CSS tab and so on.
  4. For Custom Fields, if it is your first time, you have to click on Add Custom Fields > Edit Custom Fields > paste the code in and then click Update.
  5. The Custom Fields tab should now look like the image shown below.
  6. You can modified each options in Custom Fields tab to customize your chatbox visual.
    7. Tips: You can add transparency to color by using 8 digits hex color.
  7. "Message Background Image" option is required. Recommended size is 200px height and 400px wide. Image lower than 200KB in size is recommended.
    8. You might experience unresponsive page when uploading image or selecting image. Simply wait for a bit. If the problem persist, refresh the page, your image should already be there.
  8. For "Overlay Color" option, you can skip or replace the code. Default value darken the background a bit. The color is overlayed on top of the image for readability.
  9. If you choose to change it, my recommended site is https://cssgradient.io/. The site defaulted to three color gradient, two is my preference. Please ensure opacity is not too high as it will cover/effect the background image. Copy only the value generated, not the whole code as highlighted below (in blue). You can then replace the default value in "Overlay Color" 's text-field.
  10. Click Save Settings to save your theme.
  11. Finally, you can copy the widget url located on top of the page and use it as browser source in your OBS.
  12. For testing purposes, you can add ?simulate=1 after your widget url to simulate users.
  13. Example: https://streamlabs.com/widgets/chat-box/v1/XXXCODEXXXXX?simulate=1
